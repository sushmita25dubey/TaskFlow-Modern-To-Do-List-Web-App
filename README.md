# ✅ TaskFlow - To-Do List App

A modern and responsive To-Do List web application built using **HTML, CSS, and JavaScript**. TaskFlow helps users organize daily tasks with a clean interface and smooth user experience.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 🔍 Search tasks
- 🎯 Filter tasks
- 📅 Due date support
- 💾 Local Storage
- 🌙 Dark/Light Mode
- 📱 Fully Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
